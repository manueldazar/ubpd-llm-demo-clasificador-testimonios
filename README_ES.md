# Clasificador LLM de Testimonios – UBPD

Este proyecto implementa un clasificador basado en modelos de lenguaje (LLM) guiado por una ontología YAML para analizar testimonios en contextos de derechos humanos y búsqueda de personas desaparecidas.  
Produce salidas JSON deterministas mediante un pipeline modular (preprocesamiento, construcción de prompts, validación).  
Incluye documentación bilingüe, diagramas Mermaid, API, arquitectura y una maqueta interactiva del clasificador.
