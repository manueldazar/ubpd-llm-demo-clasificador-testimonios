# UBPD LLM Testimonial Classifier – Full README

## Overview
A complete YAML‑guided, ontology‑driven LLM classifier for extracting structured information from testimonial documents.  
Includes preprocessing, prompt engineering, deterministic JSON validation, and extensible architecture.

## Components
- Preprocessing pipeline  
- Ontology loader  
- Prompt generator  
- LLM invocation client  
- JSON validator  
- Output router
